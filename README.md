## Learning Principles

1. Use official courses, documentation, and source code as the main learning path.
2. Try independently before requesting AI hints.
3. Record real errors, hypotheses, tests, root causes, and fixes.
4. Prefer small, runnable, and explainable outputs over passive information intake.
5. Rebuild core tasks without AI to verify independent ability.
